Easily find the Site Id for Multisite blogs.

== Description ==

Displays the Site Id (Blog Id) on the Network Sites page. This makes it easy to find the Site Id for each site in your Multisite install without looking in the database or editing code.

== Installation ==

1. Upload the `ms-blog-ids` to the `/wp-content/plugins/` directory
1. Network activate the plugin through the Network Plugins menu in WordPress

== Frequently Asked Questions ==

= Where do I find the Id's? =

Navigate to the Network Admin menu. Then navigate to the Sites menu. You should see a list of all your sites that are part of your Multisite install. This is where the Id's will appear.

= Why is there no Network Admin menu? =

Your WordPress install must be a Multisite install or this menu won't show up and you don't need this plugin.
